```go
if BuildMode {
    must(doBuild())
    if !RunAfterBuild && !hasAnalysisIntent() {
        os.Exit(0)
    }
}

func hasAnalysisIntent() bool {
    if TarPath != "" || Pid != "" {
        return true
    }
    return IsPostman() || IsPostmanProfile() || IsStandAlone()
}
```

3. 只有 `./maze --build` 这种单独编译的场景才需要 `--run`，但这种用法很少见

删除 `--run` 选项，简化逻辑为：

```go
    }
}
```
