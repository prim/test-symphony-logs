## Summary
实现了 VM 门面化重构：新增 `common/vm` 门面层、`shared` 共享层以及 `v1`/`v2` 两套版本实现，并将 `common/vm.go` 收敛为兼容转发层。

## Changes
- `common/vm.go`: 改为兼容层，负责 hooks 注入、默认模式选择和运行时切换转发
- `common/vm_helpers.go`: 保留 common 层的 VM 兼容辅助函数
- `common/vm/facade.go`: 新增稳定门面与版本绑定逻辑
- `common/vm/types.go`: 新增 VM 类型与 shared 类型别名
- `common/vm/select_*.go`: 新增 build tags 选择与冲突保护
- `common/vm/shared/helpers.go`: 新增共享层，承载 ELF/共享库/段查找/静态内存读取
- `common/vm/v1/vm.go`: 新增 LRU 版本实现
- `common/vm/v2/vm.go`: 新增 FULL 版本实现
- `common/reuse.go`: 兼容新的 VM 当前类型读取
- `dev-log/2026-03-14-vm-facade-refactoring-plan.md`: 勾选已完成的计划项
- `dev-log/2026-03-15-vm-facade-implementation.md`: 新增实现记录
