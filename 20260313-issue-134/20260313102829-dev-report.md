## Summary
完成 MPM 门面化重构，将共享定义、V1、V2 与默认选择逻辑拆分到 `common/mpm` 新目录；同时修复旧 `common` 包下依赖旧内部实现的测试，使其改为通过新门面/版本包验证行为。

## Changes
- `common/mpm/facade.go`: 提供稳定门面 API、共享类型别名与 V2 管理器构造入口
- `common/mpm/select_default.go`: 默认绑定 V2 实现
- `common/mpm/select_v1.go`: 提供 `mpm_v1` build tag 绑定逻辑
- `common/mpm/select_v2.go`: 提供 `mpm_v2` build tag 绑定逻辑
- `common/mpm/shared/types.go`: 抽出共享常量、`MemoryPiece`、`MemoryPage`
- `common/mpm/v1/mpm.go`: 承载 V1 版本实现与全局状态
- `common/mpm/v2/mpm.go`: 承载 V2 版本实现与关键能力（跨页 piece、known bitmap、非空页迭代）
- `common/mpm/facade_contract_test.go`: 新增门面契约测试
- `common/mpm/v2/race_test.go`: 新增 V2 known bitmap 并发冒烟测试入口
- `common/mpm/benchmark_smoke_test.go`: 新增门面 benchmark 冒烟入口
- `common/mpm.go`: 收敛为向 `maze/common/mpm` 转发的兼容层
- `common/mpm_switch.go`: 移除环境变量切换逻辑，改为门面转发
- `common/mpm_v2.go`: 收敛为兼容类型别名与构造入口
- `common/mpm_v2_test.go`: 改为通过 `common/mpm/v2` 管理器验证跨页拆分行为
- `common/mpm_v2_race_test.go`: 改为通过 `common/mpm/v2` 直接验证并发行为
- `common/mpm_benchmark_test.go`: 改为显式引用 `common/mpm/v1`、`common/mpm/v2`，避免依赖已删除的旧内部符号
