## Background
Many upper-layer modules still hand-roll the same local ResultItem aggregation pattern:

- create local `map[uint64]*ResultItem`
- build `func(s uint64) *ResultItem` via `GetResultItem`
- run scan/classify logic

- replace hand-written local ResultItem aggregation shells with `AddResult` in serial call sites
