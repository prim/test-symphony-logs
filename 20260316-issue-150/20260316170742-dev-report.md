## Changes

- `common/vm/v3/vm.go`: 为 `runtimeState` 增加统一的 in-flight 读者计数协议，不再只依赖 `closer` 保护关闭期；`Close()` 在释放 VM 前等待公共读取完成；为大 core 预加载 segment 增加 `preloaded` 快路径缓存，避免热路径重复走 atomic 指针解引用。
- `common/vm/v3/nocopy.go`: 精简 L1 cache 的 load/store 分支，减少缓存命中时的额外 `contains()` 检查和重复原子读取。
