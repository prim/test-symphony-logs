**File**: `common/vm/v3/nocopy.go`

**File**: `common/vm/v3/pagetable.go`

### 3. L1 Segment Cache (Enhancement)

### 4. Batch Read API (Scan Optimization)

- `GetMemoryBatch8(addresses []uint64, results []uint64) int`
- `GetMemoryBatch4(addresses []uint64, results []uint32) int`
