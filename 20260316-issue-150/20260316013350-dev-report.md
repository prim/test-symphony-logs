## Self-Review 发现与修复
- 发现 facade 仍通过共享状态暴露当前 VM 类型，测试与调用侧读取缺少原子语义；已统一改为 `CurrentVMType()`
- 发现 V3 快路径对未对齐地址仍可能执行 `unsafe` 直接解引用；已增加真实指针对齐校验并在不满足时走字节序读取
- 发现批量读取 API 只是标量循环；已增加排序和 segment 窗口复用，减少重复定位
- 发现 compare suite 缺少 `GetMemoryRange`、`ColdStart` 的 V3 映射；已补齐 benchmark
- 发现页表在全线性 span 场景下仍会构建；已改为仅在无法启用线性 span 时构建页表
