## Summary
修复了 VM V3 在 code review 中指出的两个主要问题：
- 将 `vmCache` 从复用整棵运行期 `*VM` 改为缓存只读模板，并在 `prepareVM()` 中返回新的运行实例，消除跨生命周期共享对象的问题。
- 在 `Close()` 中显式释放 linear span、segment data、L1 cache 与 source 引用，避免最近一次 core 的数据长期保活。

另外为 batch API 增加了已排序输入快路径与工作区复用，降低热路径中的重复分配与排序成本。

## Changes
- `common/vm/v3/vm.go`: 新增 `vmTemplate` / `segmentLayout`，重构 `prepareVM()`，补充 `release()` 释放流程。
- `common/vm/v3/batch.go`: 增加 `sync.Pool` 工作区池、排序复用逻辑、已排序输入快路径。
- `common/vm/v3/v3_contract_test.go`: 新增实例隔离、关闭释放、batch 快路径测试。
- `dev-log/2026-03-16-vm-v3-fix-iteration-5.md`: 记录本轮修复设计与验证结果。
