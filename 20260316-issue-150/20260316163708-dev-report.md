## Summary
PARTIAL：本轮修复了 V3 公共读取路径在 `closer == nil` 场景下的关闭期读者保护协议，并收紧了 `currentReadState` / `acquire*ReadState` 优先绑定 `defaultState` 的逻辑，避免 `readState` 残留快照绕过最新生命周期状态。同时尝试将页表子表改为定长直索引结构以缩短 lookup 热路径。

## Changes
- `common/vm/v3/vm.go`: 将公共读取路径的 in-flight reader 计数统一为无符号原子计数，关闭时对所有 source 路径一视同仁等待读者退出；读取时优先从 `defaultState` 获取最新 runtime state，避免并发关闭窗口。
- `common/vm/v3/pagetable.go`: 将 `subPageTable` 从稀疏切片+base 偏移改为定长 1024 槽位数组，去掉 lookup 时的 base/len 分支，简化 page table 子表访问。
