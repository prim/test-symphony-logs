## Summary
修复了 `common/vm/v3` 在 `closer == nil` 时公共 fast path 与 `Close()` 并发的生命周期漏洞，统一了大 core / 非 mmap 路径与 mmap 路径的 in-flight reader 关闭协议。

## Changes
- `common/vm/v3/vm.go`: 删除 `closer == nil` 特判，让公共读取路径统一走 `readers` 保护；`Close()` 无论 source 类型都等待在途读者退出后再释放 VM。
- `common/vm/v3/v3_contract_test.go`: 新增 `closer == nil` 路径的并发关闭回归测试，覆盖 `GetMemory`、`GetMemorySafe`、`GetMemory4`、`GetMemory4Safe`、`GetMemoryRange`。
- `dev-log/2026-03-16-vm-v3-qa-fix-iteration-38.md`: 记录本轮修复与验证结果。
