## Summary

本轮修复了 code review 指出的两个 major 问题：
- 补全 VM facade 的关闭语义，`CloseCurrent()` / `Close()` 现在会清空当前绑定快照与 `VMType`，避免“已关闭但仍被视为当前实现”的状态残留。
- 让 V3 小 core 的真实 `Init()` 路径在可用时实际接入 `shared.OpenMmap()`，使用 `mmapSource + closer`，使关闭保护抽象与生产初始化路径一致；大 core 仍保留 fileSource 快路径以满足性能目标。
