## Summary
修复了 VM V3 模板页表实例化时共享二级页表对象的问题，避免不同 VM 实例互相污染页表槽位；同时把公共 `GetMemory*` / `GetMemoryRange` fast path 纳入 inflight 生命周期协议，防止 `Close()` 并发释放底层资源时出现悬空读取窗口。

## Changes
- `common/vm/v3/pagetable.go`: 新增页表/子页表按实例深拷贝逻辑，保持模板缓存只读。
- `common/vm/v3/vm.go`: `instantiate()` 改为基于模板生成实例私有页表；公共读取 API 先获取受 inflight 保护的读状态，再执行 fast path。
- `dev-log/2026-03-16-vm-v3-fix-iteration-40.md`: 记录本轮修复背景、实现与验证结果。
