## Summary
修复了 `common/vm/v3` 在 `-race/checkptr` 下的并发首访读路径 unsafe 指针缓存问题，改为基于已加载 `[]byte` 直接读取，避免跨分配边界缓存裸指针导致的 `checkptr` fatal。复测了 V3 公共初始化、竞态测试、关键性能基准以及 Python / Node.js 主链路回归。

## Changes
- `common/vm/v3/vm.go`: 删除 `Segment.basePtr` 裸指针缓存，新增基于 `[]byte` 的 `fastData()` / `loadUint64At()` / `loadUint32At()` 读路径，修复 `go test -race` 下的 checkptr 崩溃。
