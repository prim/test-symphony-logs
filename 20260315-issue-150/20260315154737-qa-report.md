## 失败的 Test Case

### TC-005: V3 与 V2 对比基准存在且可执行

- **实际行为**: V3 在全部已实现对比场景中慢于 V2，但测试仍整体 PASS，因为缺少性能比较断言。
- **复现步骤**:
  1. 执行 `go test -tags vm_v3 ./common/vm -run '^$' -bench 'BenchmarkCompareV3(GetMemory8|GetMemory4|SequentialScan|RandomAccess|ConcurrentAccess)$' -benchmem -benchtime=20ms`
  2. 观察各子 benchmark 中 `v2-full` 与 `v3` 的 `ns/op`
  3. 再阅读 `common/vm/bench_compare_test.go`，确认仅有 `TestV3CompareBenchmarksPresent`

- **建议修复**: 优先降低段定位/边界处理慢路径开销；随后在对比测试中加入稳定、可失败的性能比较校验，至少覆盖 issue 明确要求的主场景。
