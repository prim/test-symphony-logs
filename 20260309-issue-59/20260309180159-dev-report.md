## Self-Review 发现与修复
- 发现：`collector_test.go` 仍按旧 `Merge(local, keepTopN, mergeBySize)` 签名调用，导致整个 `common/results` package 无法编译测试。
  修复：改为使用 `Configure(Config{...})` 设置策略，再调用新的 `Merge(local)`。
- 发现：`txos/results.go` 仍依赖已删除的旧全局排序状态 `PriorityQueue` / `CurrentObjectSizeMap`。
  修复：在文件内实现基于 `ObjSize` 的局部优先队列，改为显式传递对象大小 map。
- 发现：`InitAllMemoryPieceSize()` 仅重置 `ResultView` 会造成新 collector 底层状态与展示视图重置路径不一致。
  修复：统一改为 `InitAggregatedResults()`。
