## Summary

实现了 `common/mpm/v2` 热路径第二轮优化：在 `Finalize()` 后冻结页视图并将查询路径改为无锁读取，为 `IterPage()` 增加 small-page 串行 fast path，并将 `Collect()` 改为仅对跨页 piece 做局部拆分以减少中间分配。另新增正式 dev-log，记录设计、实现与验证结论。

## Changes

- `common/mpm/v2/mpm.go`: 新增冻结页快照、Finalize 后结构写入保护、无锁查询路径、`IterPage()` small-page fast path、`Collect()` 中间分配优化。
- `dev-log/2026-03-13-mpm-v2-hot-path-optimization.md`: 记录本轮热路径优化方案、benchmark 结果与自审说明。
