**[STATUS: FAIL]**

- **可能原因**: Go prepare 在深层结构采集、结构展开策略或类型记录细节上与 legacy Python 实现存在偏差。
- **建议修复**: 以 QA 基线对比失败路径为入口，对照 legacy 输出逐字段校准 `deep_loop_structs` 相关准备逻辑，再复测完整 JSON 等价性。
